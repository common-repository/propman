== Description ==

Easily Manage Your Online Real Estate For Sale & Rental Listings Web Pages from Anywhere Using a Small Yet Powerful Set of Wordpress Custom Post Types and various jQuery effects and Google Maps.

Works in a modular way to run custom post types through a plugin instead of a theme. now you can use any theme instead of being locked to a theme.

== Installation ==

1. Upload propMan to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
